# Smart Calculator

Program works as a console calculator. It can solve complicated equations with variables. 
Code uses postfix notation and stacks with priority to get a correct result.

```
> a = 2^(3+2) ++ 5
> a
37
10/(a-16) -- 2
> 3
> aa
Unknown variable
> 2+2*2*2-2
6
> b = 10
> a * b
370
> 123.23 + (-23.23) * 2
200.00
> var = 20.0 + 5.6 / 2
> var
22.8
> var = var - (3+2*3.2)
> var
13.4
> c === 2
Invalid identifier
> /help
The program calculates equations
> /exit
Bye!
```
